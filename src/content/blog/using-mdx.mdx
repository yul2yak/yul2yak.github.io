이 테마는 `astro.config.mjs` 구성 파일에 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 통합이 설치 및 구성되어 있습니다. MDX를 사용하지 않으려면 구성 파일에서 통합을 제거하여 지원을 비활성화할 수 있습니다.

## 왜 MDX인가?

MDX는 JavaScript 및 JSX 구문을 지원하는 Markdown의 특별한 변형입니다. 이를 통해 [JavaScript와 UI 컴포넌트를 Markdown 콘텐츠에 혼합](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro)하여 대화형 차트나 경고와 같은 기능을 사용할 수 있습니다.

기존 콘텐츠가 MDX로 작성된 경우, 이 통합을 통해 Astro로의 마이그레이션이 쉬워집니다.

## 예시

다음은 MDX 내에서 UI 컴포넌트를 가져와 사용하는 방법입니다.  
브라우저에서 이 페이지를 열면 아래 클릭 가능한 버튼을 볼 수 있습니다.

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	MDX에 포함된 컴포넌트
</HeaderLink>
